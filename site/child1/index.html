Community Content

This content isn't part of the main site, it's maintained by a community content editor. When changes to this site are pushed to github, a CodePipeline job is executed to build and deploy a new container to AWS ECS.

In this container, I've configured apache "AllowOverride" to "All". This gives the content editor basic control over access to the content they present.
